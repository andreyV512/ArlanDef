Инструкция как настроить группу прочности с помощью MS-DOS проги.

1.Cоздадим .txt файл вида:
	
	1 2150 2159 2185 2228 2321 2321 2257 2194 1

Где 1 это номер строки по порядку, потом 8 параметров, и группа прочности K = 0, D = -1, E = 1
Параметры берутся из базы данных, желательно заносить данные как на теплом, так и на холодном модуле 
Для того чтобы не сбивать пальцы вписывая в файл данные, открываем в Дефектоскопе: Настройки->Просмотр эталонов.
Выделяем нужные нам файлы, жмем ПКМ -> Экспорт в .txt

2.Открываем MS_DOS прогу и жмем Настройка->Коэффициенты, выбираем созданный нами файл и жмем открыть(именно Open и не щелчек 2 раза по имени)
Жмем кнопку Ввести, должно появиться примерно следующее:
	0 1 2 7 |    4 2108 2106 2113 2137 2274 2259 2200 2130    0|0.84239206115|  6
	0 1 2 6 |    1 2150 2159 2185 2228 2321 2321 2257 2194    1|0.98402982069|  3
Строк может быть и больше двух.

3. Для тестирования, после ввода коэффициентов, незакрывая программу жмем Настройка->Тестировать. Нажимаем Ввести, и на экране появляется результат:
	В заголовке пишется ГП описаная в файле, а в низу окна ГП посчитаная программой.Жмем Ввести для считывания следующей строки.
	Программа нацелена на определение групп K и E, если не К и не Е значит D

4. Для того чтобы это все заработало на дефектоскопе необходимо:
	4.1. Взять файл INI.DAT сформированный программой MS-DOS рядом с exe.шником после ввода коэффициентов.
	4.2. Скопировать этот файл в Defectoskope/Settings/
	4.2. Переименовать данный файл по принципу типоразмер.txt (например: 60.txt, 73.txt,89.txt) это нужно чтобы связать данные с текущим диаметром выставленным в программе.
	4.3. Проделать пункты 1-3 для всех типоразмеров.
	4.4. В программе Дефектоскопии, зайти в Настройки измерений->Прочие настройки, выставить "Рассчет ГП с помощью корреляции"
	4.5. Гнать эталоны и тестировать, гп будет выводиться как обычно.
	
Букву с группой прочности возвращает функция calcSolidGroup(вектор из 8ми координат) описаная в классе Tube.
Рассчет карреляции, вычитка параметров, и т.д. инкапсулированы в Private функции класса.

В ситуации если данные снимаются в отрицательном полупериоде синусоиды, (8 параметров отрицательные), то для тестирования следует применять написанную мною прогу NewSolidGroup.
Программа трехкнопочная:
1. "Открыть файл с порогами" это кнопка для открытия ini.dat файла сформированного ms-dos,овской прогой для данного типоразмера.
2. "Открыть исходный сигнал"  - выбрать .txt файл с исходными параметрами, который создается в пункте №1
3. Расчет  - считает все, тут немного не удобно, приходится открывать исходный файл и сравнивать правельно ли было посчитано.
